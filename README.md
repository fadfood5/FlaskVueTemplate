# FlaskVueTemplate

### Requirements

| software | version |
|----------|:-------:|
| nodejs   | 9.8.0   |
| python   | 3.6.4   |
| npm      | 5.7.1   |

### Install javascript dependencies
```
npm install
```

### Install python requirements
```
pip install -r requirements.txt
```


### Run backend flask app in a separate terminal
```
python run.py
```

### Run frontend with autorefresh
```
npm run dev
```

### Build for prod
```
npm run build
```
